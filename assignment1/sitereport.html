SITE REPORT

 
This project is based on HTML and CSS to make a web site. this website contains of 5 pages which is the index profile , contact, portfolio work experience and site report.
I have made this website with only html and css.
For my pages, I have linked my css and other pages and used responsive navigation bar.
 
For index page i have used the background color ad bisque, div, navigation and linked with "https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta2/css/all.min.css"
 
For my second page, Contact page, I linked it with the same CSS used the div, label for the contacting list.
 
For the third page, Portfolio page, I simply used the images from pinterest(https://www.pinterest.com/),used the same CSS.
 
For the fourth page, Work Experience page, I putted a profile used div navigation bar. I used the div, navigation, and the same css. I shared my personal information,
work experience, education and additional activites.
 
For the fifth page, Site report page, I have used the div and navigation bar and linked with same CSS.
